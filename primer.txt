\title ПРОВЕРОЧНЫЙ ТЕСТ
  Название теста

\time_all 50
  Время, данное на один вопрос (время указывается в секундах)

\number_of_attempts 10
  Количество попыток, которое будет дано на прохождение теста

\random_mode 0
  Если 1, то вопросы будут распределяться рандомно
  Если 0, то вопросы будут в том порядке, в котором они записаны

\number_of_questions 4
  Количество вопросов,  которое будет в тесте(необходимо при \random_mode 1)

\3 1
\4 2
\5 3
 Кочество правильных ответов на определенную оценку
 Если 3 = 0, 4 = 0, 5 = 0, то проверки результата теста не будет 
 Если указано значение только для 5, то тест проверяется на (Сдал/Не сдал)


\q Вопрос, ответ на который необходимо написать

\q Вопрос, ответ на который необходимо написать (на ответ дано 100с)
\time 100
  Время, данное конкретно на этот вопрос

\q Вопрос, ответ на который необходимо написать (с картинкой)
\pictures KCHVTUIOBPFASTIHVYAR
  Код для картинке можно получить, отправив картинку боту (Управление тестом -> Добавить картинку)

\q Вопрос с несколькими вариантами ответа (правильный 2,3,6)
\- вариант ответа 1
\+ вариант ответа 2
\+ вариант ответа 3
\- вариант ответа 4
\- вариант ответа 5
\+ вариант ответа 6
\- вариант ответа 7

\q Вопрос с одним вариантом ответа (правильный 2)
\- вариант ответа 1
\+ вариант ответа 2
\- вариант ответа 3
\- вариант ответа 4
